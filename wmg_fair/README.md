# Walkabout Minigolf Poptracker Pack for Archipelago

[Poptracker](https://github.com/black-sliver/PopTracker) pack for the [Archipelago Walkabout Minigolf Manual Multiworld randomizer](https://github.com/energymaster22/walkabout-mini-golf-ap-manual)

This pack was started with the help of the [ap-manual-to-poptracker](https://github.com/Ars-Ignis/ap-manual-to-poptracker) program.

## Quickstart Guide
- Download [Poptracker](https://github.com/black-sliver/PopTracker/releases)
- Grab the latest release in [the releases page](https://github.com/energymaster22/walkabout-mini-golf-ap-manual/releases) OR download this folder as a .zip file 
- Place this .zip into Poptracker's 'packs' folder
- Launch Poptracker, select the Walkabout Minigolf pack
- Connect to Archipelago by using the AP button
- Enjoy!
